# Getting Started

> 给第一次接触 LabAgent 的人。先建立整体模型，再按需要进入其他目录；不要一开始读历史日志或所有 benchmark。

## 最短路径

1. 读 [ONBOARDING_GUIDE](ONBOARDING_GUIDE.md) 的总图、第一站和第二站。
2. 按需查 [Tech Stack Knowledge Base](Tech_Stack_Knowledge_Base.md) 中不熟悉的术语。
3. 回到 [文档门户](../README.md)，按当前角色选择 architecture、operations、engineering 或 quality。

## 完成标准

- 能说清模型为什么在本地 GPU、网关为什么在云端。
- 能区分 qwen-agent、embed-local、vision-local 和 labagent-agent。
- 知道当前状态以根目录 README 和 HANDOFF 为准。

## 下一步

- 要启动或排障：进入 [operations](../operations/README.md)。
- 要理解系统设计：进入 [architecture](../architecture/README.md)。
- 要读源码：进入 [engineering](../engineering/README.md)。
